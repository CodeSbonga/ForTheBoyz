Title: Telkom Gamified Learning & Parental Dashboard – Full Stack Hackathon Build
Description: Create a modern full-stack web application that mirrors Telkom’s MyTelkom app UI/UX. The app has two parts:
1. Part A – Parental Dashboard: Allows the administrator (parent) who buys Telkom data to manage how family members use data. The admin can toggle study mode, whitelist educational sites, blacklist entertainment apps, and view detailed usage reports. It also supports different user roles: Parent (admin), Student, and Elderly (18+ who can self-restrict).
2. Part B – Gamified Learning App: Provides a React quiz game for students. It tests general knowledge, displays the score, and stores results in the database.
The system must strictly follow Telkom’s branding, UI, and design patterns (blue-and-white theme, Telkom fonts, clean layout).
